Anleitung & Setup

1. JSON Datei importieren

Lade eigene Szenarien hoch. Die Datei muss eine Liste (Array) im JSON-Format sein.

[ "Szenario 1: Dunkler Wald...", "Szenario 2: Leerer Aufzug..." ]

2. Direkter URL-Link

Willst du einen bestimmten Text üben, ohne dem Zufall zu vertrauen? Häng ihn als Parameter an die URL an. Das überschreibt den normalen Pool.

.../index.html?story=Du wachst in einem Sarg auf

Tief ausatmen.
